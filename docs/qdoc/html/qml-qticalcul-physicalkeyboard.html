Detailed Description

Les touches gérées sont les touches du pavé numérique (0 à 9), les flèches pour se déplacer entre des afficheurs d'un même opérande, Tab pour passer d'un opérateur à l'autre, Plus et Moins pour les opérations.

On gère aussi le point, delete, Ctrl-C et Ctrl-V.

La touche k (spy) pour voir le contenu du tampon.
