[static] int Converter::HMSItoMicroseconds(int, double)

Convertit un HMSI (même incomplet) en un nombre de micro-secondes.

This function returns the value converted in microseconds.
Exemple: "02:45:00:.." pour 2h45m en 9900000000 us.
hmsi : HMSI string to convert.
framerate : le framerate utilisé dans le hmsi.

[static] int Converter::addSpaceSeparator(int)

Ajoute des séparateurs de milliers sur la partie entière et la partie décimale.

value : a numeric value.
Returns a readable string with space separators.

[static] int Converter::completeRawHMSIWithDots(int)

Transforme "123" en "12:3.:..:..".

raw_hmsi : raw hmsi to convert.
Returns the hmsi formatted with : and dots.

[static] double Converter::getNtscFramePerSec()

Retourne le framerate courant du NTSC. @return the current NTSC framerate.

[static] int Converter::microseconds(int)

Transforme une valeur numérique (ex: 1'234'000'000) en un texte (ex: "1234000000"). @param microsecs : microsecs value to convert. @return a readable string

[static] int Converter::microsecsToDHMSM(int)

Transforme 1'234'000'000 en "0 days 00:20:34,000". param microsecs : microsecs value to convert. Returns a readable HMSI value with days ("N days HH:MM:SS:II").

[static] int Converter::microsecsToFrames(int, double)

Convertit un nombre de microsecondes en un nombre de frames (selon un certains framerate)

Note: Ex: 1'234'000'000 ms en 25 fps donne "30850" frames. Ex: 1'234'000'000 ms en 50 fps donne "61700" frames. @param microsecs : microsecs value to convert. @param framerate : framerate to use for conversion (ex: 25.0) @return the value converted in frames, as a string.

[static] int Converter::microsecsToHMSI(int, double)

Transforme 1'234'000'000 en "00:20:34:00". param microsecs : microsecs value to convert. param framerate : the framerate to use for the 2 last digits. Returns a readable HMSI value (HH:MM:SS:II).

[static] int Converter::microsecsToHMSM(int)

Transforme 1'234'000'000 en "00:20:34,000". @param microsecs : microsecs value to convert. @return a readable HMSm value (HH:MM:SS,mmm).

[static] int Converter::microsecsToMillis(int)

Transforme 1'234'000'000 en "1234000". @param microsecs : microsecs value to convert. @return the value converted in milliseconds, as a string.

[static] int Converter::microsecsToRawHMSI(int, double)

Transforme 1'234'000'000 en "00203400". param microsecs : microsecs value to convert. param framerate : framerate for raw hmsi Returns a raw HMSI value ("HHMMSSII").

[static] int Converter::microsecsToSeconds(int)

*************************************************************************

Transforme 1'234'000'000 en "1234".

Note: Autant que possible, on évite les divisions int/float qui introduisent des pertes de précision. @param microsecs : microsecs value to convert. @return the value converted in seconds, as a string.

[static] int Converter::rawHMSItoMicroseconds(int, double)

Convertit un Raw HMSI (même partiel) en un nombre de micro-secondes.

Returns the value converted in microseconds.
Exemple: "0245" pour 2h45m = 9 900 000 000 us.
hmsi : raw hmsi to convert.
framerate : the framerate used for raw hmsi.

[static] void Converter::setNtscFramePerSec(double framerate)

Change le framerate du NTSC. framerate : the new NTSC framerate.

[static] int Converter::toMicroseconds(int, double)

Transforme une unité quelconque (string) en microsecondes (numerique).

Note: Hors HMSI. param value: un nombre d'unités (string). param conversionFacteur: le facteur de conversion de cette unité. Retourne un nombre de microsecondes. -1 en cas d'erreur.
