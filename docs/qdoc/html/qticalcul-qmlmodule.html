TiCalcul QML Module

Description des types QML définis pour TiCalcul.
La description des classes C++ se trouve sur cette page.

Liste des Types QML :

AboutPopup

Ce popup affiche l'Auteur, l'Année et le Numéro de version de l'application

ActiveDisplay

Mémorise l'afficheur actif

Bloc

Un petit rectangle, visible ou non à l'écran, pour remplir un espace

CalculFrame

Controlleur pour le panel central (zone des calculs)

CalculFrameForm

Affiche les Operande en HMSI ainsi que le résultat du calcul, une imagette et un éventuel message

FieldInput

Mémorise l'afficheur actif, lors d'un click sur un afficheur FieldInput

FieldInputForm

Le FieldInputForm est associé à un objet C++ Afficheur

FieldLabel

Affiche un libellé à coté de chaque afficheur QTiCalcul::FieldInput. Il faut fournir la property text

FieldResult

Gère un champ numérique de l'IHM

Keyboard

Le Keyboard est composé d'un ensemble de boutons KeyboardKey

KeyboardForm

Le KeyboardForm est un ensemble de 18 boutons de type KeyboardKeyForm

KeyboardKeyForm

Fournit un bouton customisé pour les touches du clavier virtuel

LeftFrame

Controlleur pour le panel de gauche

LeftFrameForm

Panneau de gauche, avec l'affichage du Timecode TC1 dans différentes unités

Main

Fenêtre principale

PhysicalKeyboard

Gestion des touches du clavier

RightFrame

Controlleur pour le panel de droite

RightFrameForm

Panneau de droite, avec l'affichage du Timecode TC2 dans différentes unités

SquareButton

Petit bouton carré. Fournit le texte et la couleur

TiButton

Petit bouton arrondi

TopBar

Controller pour la barre du haut, qui contient le bouton pour AboutPopup
