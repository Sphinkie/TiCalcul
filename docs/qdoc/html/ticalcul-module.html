Detailed Description

The ultimate Timecode Calculator.

  • Les classes C++
  • les scripts QML

TiCalcul est developpé en C++ et en QML (avec le framework Qt version 6).
Il lit et écrit les tags EXIF et IPTC des photos JPEG avec exifTool.
Il accède aux cartes géographiques via les modules QtLocation et QtPositioning.
