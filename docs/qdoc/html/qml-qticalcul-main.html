Detailed Description

La fenêtre principale contient 3 zones: LeftFrame, CalculFrame et RightFrame. Au dessus, se trouve la TopBar avec le bouton pour faire apparaitre le AboutPopup. En dessous, on a le clavier numérique Keyboard.

See also ActiveDisplay, PhysicalKeyboard, and Keyboard.
