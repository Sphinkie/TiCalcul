Detailed Description

Member Function Documentation

[explicit] Operande::Operande(QString name, QObject *parent = nullptr)

Constructeur. Crée les 8 Afficheur de cet opérande, et les connecte pour leur envoyer un signal à chaque modification de la valeur pivot.

Note: L'afficheur NTSC est désactivé pour le moment.

name: Le nom de l'operande: "tc1" ou "tc2" ou "result".
parent: Pointeur sur QObject parent.

[invokable] void Operande::clearValeurPivot()

SLOT : Vide la valeur pivot, et efface la valeur de tous les Afficheur.

Note: This function can be invoked via the meta-object system and from QML. See Q_INVOKABLE.

See also Afficheur::clearValue().

[invokable] void Operande::connectActiveDisplay(int *engine)

Cette fonction crée des connexions slot-signal.

Cree des connexions entre l'objet QML "afficheurActif" et tous les Afficheurs. Ainsi, lorsque l'Afficheur actif change, tous les afficheurs sont informés. engine: un pointeur sur le moteur QML.

Note: This function can be invoked via the meta-object system and from QML. See Q_INVOKABLE.

void Operande::registerContext(int *context)

Enregistre les Afficheur dans le contexte QML, pour pouvoir les afficher.

Note Les property names seront du type "tc1_aff_1".
context: Le rootContext de l'application.

void Operande::setPartner(Operande *partner)

La fonction setPartner recoit un pointeur sur l'objet Operande partenaire partner (=TC2), et crée un "apairage" entre chacun de ses afficheurs et les afficheurs de l'operande partenaire TC2.

[slot] void Operande::setValeurPivot(const qint64 newValeurPivot, const bool force = false)

Ce slot recoit une nouvelle valeur pivot d'un Afficheur et la propage à tous les Afficheurs.

La newValeurPivot est exprimées en microsecondes, et le paramètre force précise s'il faut la prendre en compte même si le champ est en cours d'édition.

Peut aussi être positionné par le calculateur.
Réception du signal Afficheur::setValeurPivot().
Envoie le signal Operande::valeurPivotChanged() aux Afficheur.
Le slot de réception des Afficheur est setValue()

See also valeurPivot().

qint64 Operande::valeurPivot() const

Renvoie la valeur Pivot (en microsecondes).

See also setValeurPivot().
