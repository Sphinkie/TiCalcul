Description détaillée

La classe Afficheur gère un champ numérique de l'IHM QML.

Elle est associée à un objet QML FieldInput et lui envoie les valeurs à afficher. Il y a un Afficheur pour chaque unité: Secondes, Millisecondes, HMSI, etc.

Mémorise et propage la string à afficher vers le QML.

Note
On distingue le cas où la cellule est active (cad en cours d'édition), ou pas. Si la cellule est active, on ignore la valeur reçue et on continue d'afficher la valeur brute qui est en cours d'édition.
Paramètres
valueLa valeur brute exprimée dans l'unité de cet afficheur (rawHMSI ou rawNUM).
forcesi true, on prend en compte la valeur, meme si l'afficheur est actif.
