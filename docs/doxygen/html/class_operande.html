◆ mMaxValue

const qint64 Operande::mMaxValue = 99999999999999
private

Valeur pivot maximale autorisée (99 999 999 999 999 microsecondes) soit environ 1000 jours. A noter que le uint32 est limité à 4.294.967.295. D'où l'utilisation du qint64 (long long)
